# waiting
waiting service for mobile pwa version

## Demo
https://ann8355.github.io/waiting/index.html

## Build Setup

``` bash
# yarn build Vue CLI 3
yarn global add @vue/cli

# 保留舊版的vue cli
yarn global add @vue/cli-init

# npm build Vue CLI 3
npm install -g @vue/cli

# 開新專案
vue create my-project

# Run project
yarn serve

# Build project
yarn build
```
